# AutoService Demo

## AI-assisted landing page experiment

AutoService Demo — экспериментальный лендинг для автомобильного сервиса.

Проект создан как проба визуальной подачи, структуры лендинга и статической публикации через GitHub.

---

## Что внутри

Лендинг включает основные блоки сервисного сайта:

- первый экран с оффером;
- направления услуг;
- блок о компании;
- примеры работ;
- сезонное предложение;
- отзывы;
- гарантии;
- контактный блок;
- простой JavaScript для интерактива.

---

## Назначение проекта

Этот репозиторий не является основным направлением профиля.

Он оставлен как часть экспериментального архива: тестирование GitHub Pages, HTML/CSS/JS, визуальных идей и AI-assisted workflow.

Главные проекты профиля сейчас связаны с внутренними цифровыми инструментами, сервисными системами, заявками, учётом техники и контентными платформами.

---

## Статус

**Experimental / demo project**

Проект можно использовать как пример структуры лендинга, но он не является коммерческим продуктом или готовым сервисом.
